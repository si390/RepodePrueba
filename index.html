GIT

IMPORTANTE: Cada vez que abro la terminal me fijo en qué carpeta estoy

  1. Crear repositorio remoto
  2. Clonar repositorio remoto en la PC con su URL usando git clone
  3. Damos credenciales
  4. Verificamos el estado con git status
  5. Añadimos con git add .
  6. Con git commit -m "algo" especificamos qué cambiamos
  7. Usamos el comando git push para subir los cambios a GitHub

AVISO: git clone se usa una sola vez
